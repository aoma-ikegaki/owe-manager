# owe-manager(開発中)

「誰に、いくら、いつまでに」返すべきかを明確にし、返済漏れを防ぐための個人向け債務管理アプリケーションです。

[デモサイト] (https://42dc705f.owe-manager-web.pages.dev/)

## 技術スタック
| カテゴリ | 技術 |
| :--- | :--- |
| Frontend | Next.js (App Router), TanStack Query, Zod, Tailwind CSS |
| Backend | Hono, Cloudflare Workers |
| Database | Cloudflare D1(SQLite), Drizzle ORM |

## 主な機能
- 借金の表示・登録・編集・削除（CRUD）
- 返済のステータス更新
- 現在の合計借金額表示
- 未返済/返済済みの一覧表示
